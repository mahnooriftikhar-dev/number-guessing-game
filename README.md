# 🎯 Number Guessing Game

A simple Number Guessing Game built using JavaScript.  
In this game, the computer randomly selects a number within a given range, and the user tries to guess the correct number.

This project is great for beginners to understand basic JavaScript concepts like variables, functions, random numbers, and user input.

---

## 🚀 Features

- Generates a random number automatically
- User guesses a number within a defined range
- Checks whether the guessed number is correct
- Displays result messages to the user
- Simple and beginner-friendly logic

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

---

## 🎮 How to Play

1. Open the game in your browser.
2. Enter a number within the given range.
3. Click the **Guess** button.
4. The game will check if your guess matches the random number.
5. Keep trying until you guess the correct number.
